# Odoo Developer Kit

Odoo Developer Kit (`odk`) is a Rust CLI for building professional Odoo
development environments with a Flutter-like command-line workflow.

## Features

- Environment diagnostics for Odoo development tools.
- Interactive project generation.
- Python environments created with `uv`.
- Docker development setup.
- PyCharm and VS Code integration.
- Cross-platform support for macOS, Linux, and Windows.

## Quick Start

Install ODK:

```bash
curl -LsSf https://odk.dev/install.sh | sh
```

Check your environment:

```bash
odk doctor
```

Create a project:

```bash
odk create
```
